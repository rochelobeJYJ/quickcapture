# QuickCapture 📸

고성능 네이티브 데스크톱 캡처 애플리케이션입니다. `PyQt6` 기반의 모던한 UI와 투명 오버레이(캡처 영역) + 플로팅 미니 툴바로 화면의 원하는 영역을 단축키 한 번으로 빠르게 캡처합니다.

## ✨ 주요 기능
- **플로팅 툴바**: 캡처 📸, 폴더 열기 📂, 설정 ⚙️, 항상 위 📌 토글, 종료 ✖ 를 오버레이 위에서 바로 조작합니다.
- **실시간 크기 표시**: 툴바에 캡처될 영역의 **실제(물리) 픽셀 크기**가 항상 표시되어, 원하는 해상도로 정밀하게 맞출 수 있습니다.
- **시스템 트레이 상주**: 트레이 아이콘에서 즉시 캡처 / 오버레이 표시 토글 / 설정 열기 / 종료가 가능합니다. 트레이 아이콘 더블클릭으로 설정 창을 엽니다.
- **정밀한 캡처 (DPI-Aware)**: 윈도우 디스플레이 배율(125%, 150% 등)과 무관하게 정확한 물리 픽셀을 캡처하며, 캡처 순간 오버레이 테두리를 자동으로 숨겨 이미지에 테두리가 포함되지 않습니다.
- **전역 단축키**: 기본 `Ctrl+Shift+C`. 설정 창에서 클릭 한 번으로 원하는 조합으로 변경할 수 있고, 오버레이 영역 **더블클릭**으로도 즉시 캡처됩니다.
- **유연한 파일명 양식**: 연번+시간 / 날짜+시간 / 직접 지정 이름 3가지 양식, PNG·JPG 포맷 선택, 파일명 중복 시 덮어쓰지 않고 `(1)`, `(2)` 자동 부여.
- **캡처 피드백**: 캡처 성공 시 테두리가 흰색으로 점멸하고, 설정 창 하단 상태 표시줄에 저장된 파일명이 표시됩니다. 실패 시 원인도 표시됩니다.
- **테두리 커스터마이징**: 오버레이 테두리 색과 두께(1~12px)를 설정 창에서 변경할 수 있습니다.

## 🚀 기술 스택 & 요구사항
- **Python 3.10+**
- `PyQt6` — GUI 엔진
- `mss` — 고속 스크린샷 캡처
- `Pillow` — 이미지 저장(JPG 품질 제어)
- `keyboard` — 전역 단축키 후킹

## 🛠️ 실행 방법
### 파이썬 환경에서 직접 실행
```bash
pip install -r requirements.txt
python main.py
```
콘솔 창 없이 조용히 실행하려면 `main.pyw`를 더블클릭하거나 `pythonw main.pyw`로 실행하세요. (`main.pyw`는 `main.py`를 호출하는 진입점이므로 두 파일을 함께 두어야 합니다.)

### 단일 실행 파일(.exe) 빌드 (PyInstaller)
```bash
pip install pyinstaller
pyinstaller QuickCapture.spec
```
완료 후 `dist/QuickCapture.exe`를 바로 실행할 수 있습니다. 설정 파일(`settings.json`)은 exe와 같은 폴더에 저장됩니다.

## 📂 코어 구조
- `main.py` — 앱 라이프사이클, 설정 관리, 전역 단축키, 캡처·저장 로직, 설정 창 UI, 트레이 아이콘.
- `main.pyw` — 콘솔 없는 실행용 진입점 (`main.py` 재사용).
- `overlay.py` — 프레임리스 투명 오버레이: 이동/리사이즈, 미니 툴바, 실시간 크기 표시, 캡처 영역 계산.
- `QuickCapture.spec` — PyInstaller 빌드 스펙 (아이콘 리소스 포함).
- `icon.ico` — 앱/트레이 아이콘.

## 💡 사용 팁
- 설정 창을 닫아도 프로그램은 종료되지 않고 트레이에 상주합니다. 완전 종료는 툴바 ✖ 버튼 또는 트레이 메뉴의 **종료**를 사용하세요.
- 오버레이 가장자리를 드래그하면 크기 조절, 안쪽을 드래그하면 이동입니다.
- 저장 폴더 기본값은 `내 문서\Pictures\QuickCapture` 이며 설정에서 변경할 수 있습니다.
