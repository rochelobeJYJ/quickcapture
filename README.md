# QuickCapture Pro 📸

고성능 네이티브 데스크톱 캡처 애플리케이션입니다! `PyQt6`를 사용하여 세련되고 모던한 UI를 제공하며, 직관적인 플로팅 미니 툴바를 통해 화면을 간편하게 지정하고 캡처할 수 있습니다.

## ✨ 주요 기능
- **플로팅 툴바 제공**: 캡처, 폴더 열기, 메인 설정 호출, 그리고 `항상 위(📌)` 토글 스위치를 오버레이 위에서 직관적으로 바로 조작할 수 있습니다.
- **모던하고 유려한 디자인**: 마이크로소프트 Fluent 스타일에 영감을 받은 매끄러운 화이트/그레이톤 테마를 적용했습니다.
- **정밀한 캡처 (DPI-Aware)**: 윈도우 디스플레이 배율(125%, 150% 등)에 전혀 구애받지 않고 오차 없는 정확한 물리적 픽셀을 캡처해냅니다.
- **스마트한 단축키 및 넘버링 방어 시스템**: 키보드 후킹 스레드 방해 충돌을 차단하였으며, 캡처된 넘버링에 중복이 발생할 경우 덮어쓰지 않고 안전하게 파일을 보호합니다.
- **단독 프레임 실행**: 프로그램을 구동할 때 불필요한 빈 설정창 없이 투명한 캡처 오버레이만 곧바로 띄워집니다.

## 🚀 기술 스택 & 요구사항
- **Python 3.x**
- `PyQt6` (화일 처리 및 GUI 엔진)
- `mss` (압도적인 속도의 모니터 스크린샷 캡처엔진)
- `Pillow` (초고속 이미지 버퍼링 및 압축)
- `keyboard` (글로벌 단축키 후킹)

## 🛠️ 실행 및 시작 방법
### 파이썬 환경 직접 실행
아래 명령어를 통해 필요 모듈을 세팅하고 실행할 수 있습니다.
```bash
pip install PyQt6 mss keyboard Pillow
python main.py
```
백그라운드에서 콘솔 없이 조용하게 띄우고자 한다면 `pythonw main.pyw` 형태로 실행할 수 있습니다.

### 단일 실행 파일(.exe) 시스템 배포 (PyInstaller)
파이썬 엔진이 탑재되지 않은 일반 사용자 PC에 배포하기 위해서는 단일 파일로 묶어야 합니다.
```bash
pip install pyinstaller
pyinstaller --noconsole --onefile --icon="icon.ico" --name="QuickCapture" main.py
```
완료 후 `dist` 폴더 안에 컴파일된 `QuickCapture.exe`를 누구나 바로 클릭하여 사용할 수 있습니다.

## 📂 코어 구조 분석
- `main.py` : 프로그램의 모든 라이프사이클과 설정을 제어하는 심장부입니다.
- `overlay.py` : 실제 화면 최상단 위에서 클릭과 위치 드래그 좌표, 미니 툴바를 연산하는 프레임리스(Frameless) 액티브 윈도우 체계입니다.
- `icon.ico` : 프로그램의 얼굴을 담당하는 모던 앱 벡터 아이콘입니다.
